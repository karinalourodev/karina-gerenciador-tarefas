#  Gerenciador de Tarefas

---

##  Sobre o Projeto
Este é um **Gerenciador de Tarefas** desenvolvido em **React**, como atividade prática de programação.  
Ele permite adicionar, marcar como concluídas e filtrar tarefas, com uma interface estilizada em gradiente azul.

---

## Funcionalidades
- ➕ Adicionar novas tarefas  
- ✅ Marcar tarefas como concluídas  
- 🔍 Filtrar tarefas por **Todas**, **Concluídas** e **Pendentes**  
- 🧹 Limpeza automática do campo de texto após adicionar  
- 🎨 Estilização com gradiente azul e interface amigável
